A C T I V I T I E S

Computer Networks Term Projects

All programs should be written in C or C++ under linux or Windows. You should run and demo your program and answer questions on the project presentation date (June 12, 2013). Besides you have to submit a report describing your algorithm. 

  1. Write a program to exchange files between a server and a client. Your program should be able to get the list of the files in the working directory of the server. Client can send a file using  “put filename” command and receive a file using “get filename” command. It should also be able to get the list of the files from the server using “ls” command. Client can terminate the connection using “bye” command.   (1 student)
  2. Write a chat program between some hosts. Any host can talk to any other host through the server. No talk includes more than two clients but the server can handle several talks at the same time. The server also should have a list of current states of the clients (online, offline ) and send it to a client if the client requests for it. The client can ask for the status list by sending STATUS at the body of a message to the server.  (2 Student)
  3.   Write a program to monitor the traffic of the packets in a local area network. Your program will read an IP address from the user and grab all packets sent to that IP or coming from that IP.        (2 students)
  4.   Write a program to create and send a DNS query message. Your program should wait for the reply and print the IP address after receiving the reply. (2 students)
  5. Write a program to send an ECHO ICMP message to a given address and receive ECHO REPLY. Your program should print on the screen the echo message and echo reply times (2 student)
  6. Write a program to find out which TCP/UDP ports in a given server are open. (port scanner) (2 student)
  7. Write a program to detect DHCP servers. Your progeam should print the reply from the server including offered IP address, Netwmask, and the IP address of the server.
  8. Use sockets to communicate with a mail server. Create a mail and send it to a given address through the mail server. (2 students)
  9. Implement traceroute using sockets.
  10. Network Time Protocol (NTP) (2 students) More details can be found here

Those students who prefer to have a non-programming term project may contact me for a topic. The deadline for these projects is also June 12th, 2013. The students should prepare a report and present their research (about 10 minutes).

Presentations will be at H302
